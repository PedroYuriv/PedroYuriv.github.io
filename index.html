Habilidades tecnológicas

+

HTML 5

‘HyperText Markup Language’ es lo que significan sus siglas en inglés. En español, se traduce como ‘Lenguaje de Marcado e HiperTexto’. Este lenguaje de marcado es la herramienta que nos permite construir la estructura básica de un sitio web a través de etiquetas. El número cinco (5) indica la versión actual del lenguaje. Gracias a este lenguaje, podemos tener títulos, párrafos, formularios, videos y muchas otras cosas en nuestros sitios web.

+

CSS 3

CSS, que significa ‘Cascading StyleSheet’ en inglés o ‘Hojas de estilo en cascada’ en español, es un lenguaje de estilo. La versión actual del lenguaje es la número tres (3). Este lenguaje nos permite añadir estilos a la estructura de nuestros sitios web, actuando como el maquillaje del HTML. Todo el diseño y la apariencia de un sitio web están directamente relacionados con este lenguaje.

+

JavaScript

JavaScript es un lenguaje de programación que nos brinda la capacidad de modificar, eliminar, aplicar animaciones y efectos a los elementos de nuestro sitio web. Es la herramienta ideal para crear sitios web dinámicos y visualmente atractivos en combinación con CSS y HTML. Con el tiempo, JavaScript ha experimentado un crecimiento exponencial y se ha convertido en un conocimiento esencial para cualquier desarrollador web.

+

React JS

React JS es una biblioteca de código abierto en JavaScript que se desarrolló para facilitar la creación de interfaces de usuario, especialmente para el desarrollo de aplicaciones de una sola página. Esta biblioteca es mantenida por Facebook junto con una comunidad de desarrolladores de software libre, que cuenta con más de mil miembros. React JS está diseñado para ayudar a los desarrolladores a crear aplicaciones que manejan datos que cambian constantemente. Su filosofía es ser simple, declarativo y fácil de integrar. React JS se encarga únicamente de la interfaz de usuario en una aplicación, actuando como la Vista en el patrón Modelo-Vista-Controlador (MVC).

+

Vue JS

Vue JS, también conocido simplemente como Vue, es un framework de código abierto en JavaScript que se utiliza para desarrollar interfaces de usuario y aplicaciones de una sola página. Fue desarrollado por Evan You, quien junto con un equipo central de miembros activos de diversas empresas, continúa manteniéndolo. Vue JS se caracteriza por su arquitectura de adaptación gradual, que se enfoca en la representación declarativa y la composición de componentes. La biblioteca central de Vue JS se centra exclusivamente en la capa de vista de la aplicación.
